RAR in macOS

The Unarchiver 可以解壓縮大多數的 rar 檔案，但有少數的 rar 檔案無法解壓縮，譬如說被壓縮的檔案名編碼不是 UTF-8，該怎麼辦呢？

其實 RARLAB 官方有提供一組 command line tool 來 壓縮 / 解壓縮 rar 檔案。目前只要 Unarchiver 不能解壓縮的檔案，我都能順利用 official rar CLI 搞定。:

  • 下載 RAR 5.91 for macOS (64 bit)
tar zxvf rarosx-5.9.1.tar.gz

# You can export the path in bash/zsh
./rar/rar -e ${YOUR_RAR_FILE}
